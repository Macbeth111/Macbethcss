The second lesson discussed selectors in CSS In CSS(Cascading Style Sheets), elements are the fundamental building blocks used to style and format HTML content on a webpage.

The four main groups of selectors discussed were

  2. Class selectors
  3. ID selectors
  4. Universal selectors

Lesson Three

the third lesson ws about applying colours in CSS. Some of the methods discussed included:

  1. Named Colors where predefined names of colours are used
  2. Hexadecimal Colours where hex code (#) is used to specify colours
  3. RGB clours where colours are defined using "rgb()" function
  4. HSL colours which involves use of the hsl() function, which stands for hue, saturation, and lightness

Lessons Five and Six

Lesson five focussed on Box-Model

The instructor opened dev-tools and showed how to check the parameters of the box around the headers. the main elements thus the content, the padding, the borders and the margins that make up the box-model were discussed.

The instructor also discussed how to use CSS-Reset to take control of the box-model settings. some of the items discussed under CSS-Reset are:

  1. Content-Box: which is the default and does not include padding
  2. Border-Box: which includes the border and padding
