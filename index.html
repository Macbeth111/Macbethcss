Discussion in lesson three

the third lesson ws about applying colours in CSS. Some of the methods discussed included:

  1. Named Colors where predefined names of colours are used
  2. Hexadecimal Colours where hex code (#) is used to specify colours
  3. RGB clours where colours are defined using "rgb()" function
  4. HSL colours which involves use of the hsl() function, which stands for hue, saturation, and lightness
